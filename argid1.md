# 1. ПРЯМЫЕ КОНКУРЕНТЫ (AR-приложения)

‎
## StreetMuse
‎- Платформы: iOS, Android

‎- Модель: Бесплатное приложение

‎- Что умеют: AR-наложение исторических фото на здания в центре Москвы и Петербурга

‎- Контент: Только редакторский (наполняют сами)

‎- Покрытие: Около 300 точек, только центр, только 2 города

‎- Слабые места: 
‎  - Нет возможности пользователям добавлять контент

‎  - Нет веб-платформы для планирования маршрутов

‎  - Нет социальных функций (лайки, комментарии, избранное)

‎  - Ограничены скоростью своей редакции


## HistAR
‎Платные туры ($5-10)

‎ Что умеют: 3D-реконструкции древних сооружений поверх реальности
‎Контент: Профессиональный 

‎Локации: Европа, Ближний Восток 

‎- Слабые места:

‎  - Требует мощное железо

‎  - Нужно скачивать гигабайты данных

‎  - Нет контента для обычных городских прогулок

‎  
‎
## ARrrrr! (Германия)
‎Платформы: iOS, Android

‎Бесплатно + премиум-маршруты

‎ Что умеют: Простые AR-квесты (найти точки, собрать предметы)
‎Контент: Пользовательский (любой может создать квест)

‎Локации: Зависит от пользователей

‎- Слабые места:

‎  - Слабая AR-технология (просто метки)

‎  - Нет исторического контента (только игры)

‎  - Мало пользователей за пределами Германии

‎
‎ КОСВЕННЫЕ КОНКУРЕНТЫ (решают ту же задачу иначе)

## izi.travel
‎Процент с продаж туров, подписка

‎Что умеют: Аудиогиды по городам и музеям с GPS-триггерами
‎-Контент: UGC (тысячи профессиональных гидов и любителей)

‎Локкции: 10 000+ городов мира

‎- Сильные стороны:

‎  - Огромная база контента

‎  - Выстроенные отношения с авторами

‎  - Работающая бизнес-модель

‎  - Есть веб-редактор туров

‎- Слабые стороны:

‎  - Нет AR (только аудио)

‎  - Нет визуального взаимодействия с городом

‎  - Формат "лекции" устаревает


## Surprise Me
‎- Платформы: iOS, Android

‎- Модель: Бесплатное

‎- Что умеют: Готовые маршруты по городу (текст + фото)

‎- Контент: Пользовательский + редакторский

‎- Покрытие: Москва, Питер, Казань

‎- Слабые места:

‎  - Нет AR

‎  - Нет монетизации

‎  - Мало активных пользователей

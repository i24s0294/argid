# 1.Цели и назначение проекта

Цель проекта заключается в разработке мобильного приложения-гида, использующего технологии дополненной реальности для повышения вовлеченности пользователей, улучшения навигации и предоставления уникального образовательно-развлекательного контента в различных локациях

Основные задачи проекта:
- создание эффекта присутствия через визуализацию исторических событий;
- повышение туристической привлекательности локаций за счет внедрения игровых механик.
